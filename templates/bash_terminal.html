Bash Task Checklist (click tasks for details)

{% with checklist=checklist %}
List & locate files (ls, pwd)

Description: See what’s in your current directory and where you are.


Commands:

ls
pwd


Tip: Use these often so you always know “where” you are in the filesystem.

System info & help (help, whoami, date)

Description: Learn about commands, who you are, and basic system info.


Commands:

help
whoami
date


Tip: These are great “information” commands when you’re exploring a new system.

Move location (cd)

Description: Change which folder you are currently in.


Examples:

cd projects
cd ..
cd /home/user


Tip: Combine cd with ls and pwd to navigate confidently.

Read / write text (echo)

Description: Print or send text to the screen or to a file.


Examples:

echo "hello world"
echo "hi" > note.txt


Tip: echo is the simplest way to see variables and quick output.

Manipulate files (touch, rm, cp, mv)

Description: Create, delete, copy, and move files.


Examples:

touch notes.txt
rm notes.txt
cp source.txt backup.txt
mv old.txt archive/old.txt


Tip: In a real shell, rm permanently deletes files—be careful!

Navigation & session control (cd -, clear, exit)

Description: Quickly move back, clean the screen, and end the session.


Examples:

cd -
clear
exit


Tip: cd - acts like a “back” button, and clear keeps your view tidy.

This checklist is tied to your account – once a box is checked, it stays checked across dashboards and classes.
